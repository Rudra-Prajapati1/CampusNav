# Future / Experimental Modules

This directory contains intentionally preserved, currently inactive modules.

- `3d-navigation/`: archived renderer and indoor 3D navigation experiments.
- `sensor-fusion/`: archived motion/orientation fusion experiments.

These modules are **not part of the current production import/routing chain**.
They are retained for future CampusNav roadmap work (3D indoor rendering, sensor-guided positioning, and AR-assisted navigation).
